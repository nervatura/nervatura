Nervatura
=========

Business Management Framework

## Features

Nervatura is a business management framework. It can handle any type of business related information, starting from customer details, up to shipping, stock or payment information.

## MCP endpoints

- /mcp -> Open access entry point. Here you will find information on how to use MCP tools, resources and prompts
- /mcp/public -> Tools, resources, and prompts that can be used without user identification and privileges
- /mcp/all -> All tools, resources, and prompts in one place.
- /mcp/customer -> Tools, resources, and prompts for managing customer data: customer, contact, address, event, report
