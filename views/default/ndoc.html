  • NTOUCH Nervatura Touch
    The graphical user interface of the Nervatura Framework. A commercial product which can be downloaded to several operating systems: Android, Windows 10 and Google Chrome Extension. Touch-optimized and responsive graphical user interface for all devices (smartphone, tablet, laptop, PC).
    The program works through the Programming Interface, offers a user friendly alternative to Data Interface. Its structure covers the needs of a usual company. Besides the business management tasks it can be used for setting other parameters of the system as well. Also, it includes a built-in report editor.
    Support to Developer Mode (Chrome Extension): Enables connection to Nervatura Framework run on a localhost. Free usage is offered to developers, testers of Nervatura Framework and system administrators.
    Download links:
    • Chrome Web Store (Chrome OS, Mac OS, Linux, Windows)
    • Google Play (Android ARMv7 and x86)
    • Windows Store (Desktop and mobile)
