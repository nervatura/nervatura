Curl examples

Login to the demo database and get a token:

curl -X 'POST' 'http://localhost:5000/api/v6/auth/login' \
  -H 'accept: application/json' -H 'Content-Type: application/json' \
  -d '{"user_name": "admin", "password": "", "database": "demo"}'
