Input fields #

Type #

rate, buy(interest), average(interest), sell(interest)

Date #

Starting date.

Currency #

The elements of CURRENCY. Its default value is taken from DEFAULT SETTINGS default currency.

Value #

rate or interest value

Group #

A valid item of the group should be defined.

Account No. #

In the search field a bank account can be chosen from bank type items of PLACE.
